This is a module for authenticating a username and password against stored credentials in a MySQL database.

Please submit a set of credentials to test the module. There are 3 valid sets of credentials:
'John Doe' + 'johnspassword',
'Jane Doe' + 'janespassword',
'Third User' + 'thirdpassword'.

Using any of these credentials will result in a successful login, while any other credentials will be rejected.
